# DevOpsPulse
DevOps Pulse

A Python-based system monitoring and automation platform built to demonstrate modern DevOps, Site Reliability Engineering (SRE), and Infrastructure Operations practices.

**Current Features**

* System health monitoring
* CPU utilization
* Memory utilization
* Disk utilization
* System uptime
* Host information

**Planned Features**
* Service monitoring
* Website monitoring
* REST API monitoring
* Alerting
* Dashboard
* Logging
* Automation
* Reporting

**Technologies**
* Python
* psutil
* Git
* GitHub
